Delete PK Key

To delete an existing PK key, follow the steps:

1. Read the FD file (refer to Modify Logo step 1).

2.  Select Delete PK Key in the PK option from the Binary ROM menu . The Delete PK Key window appears.


Delete PK Key

3. Select the item to be deleted in the Key Information list box.

4. Click the Apply button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
