Insert PK Key

To insert a non-existing PK Key, follow the steps:

1.      Read the FD file (refer to Modify Logo step 1).

2.      Select Insert PK Key in the PK option from the Binary ROM menu .The Insert PK Key window shows on the rigth side.


Insert New PK Key

3.      Specify the PK Key file path and choose the correct Key listed in the Key List.

4.      Click the Apply button to start processing. The tool will show an error message dialog if failed, or display a " Success " message in status bar if successful.
