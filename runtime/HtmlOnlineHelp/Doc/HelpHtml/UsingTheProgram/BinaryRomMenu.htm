Binary-Rom

To use the Binary-Rom function, select Binary-Rom from the menu bar upon loading the file. A drop-down menu appears for you to select the option.

Replace OpROM

Selecting the Option Rom option shows a window. Through this Option Rom window, you can change the ROM.


Option Rom window

Similarly, you can select the option On Board VGA , External VGA or PXE to change the VGA driver or PXE, respectively.
Export OpROM

To export a OpROM as a file, select it by guid or by the pair of VendorID and DeviceID, and specify the output file path, and press the "Apply" button to export.


Option ROM export window

Export OpROM list


Indicate a file name and press “Apply” button to export OpROM list. Export list content appears as following:


Option ROM list file

BIOS Version

To replace BIOS version, just key in new BIOS version and new product name, then press the "Apply" button to complete operation.


Bios version window


ESRT

To replace the ESRT, just key in new ESRT version and new ESRT GUID, then press the "Apply" button to complete operation.


ESRT window


Release Date

To replace the release date, just key in new release date, then press the "Apply" button to complete operation.


Release Date window
