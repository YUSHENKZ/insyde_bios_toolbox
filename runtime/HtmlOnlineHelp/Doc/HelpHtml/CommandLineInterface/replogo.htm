replogo

      -replogo:<SrcBIOSImageName> -logopath:<LogoImagePathName> -logoindex:<Index id> -logoconvert:<0/1> [-out:<OutputPath>]

 

To replace a logo, the following commands are required:

      -logopath : To specify the logo file path.

      -logoindex : To specify the logo module's index id.

      -logoconvert : To specify whether to convert input image format to FD image format(1), or to keep input image format(0).

 

Use the following command to display available logos:

      -replogo:<SrcBIOSImageName>

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
