repallseckey

      -repallseckey:<SrcBIOSImageName> -varname:<PK | KEK | db | dbx> -seckeypath:<SecureKeyPathName> [-sigowner:<GUID>] [-out:<OutputPath>]

 

To replace all secure keys, the following commands are required:

      -varname : To specify the variable name which is one of PK/KEK/db/dbx.

      -seckeypath : To specify the key path.

 

Use the following command to display all secure keys:

      -repallseckey:<SrcBIOSImageName>

 

The following command is optional:

      -sigowner : To specify the GUID.

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
