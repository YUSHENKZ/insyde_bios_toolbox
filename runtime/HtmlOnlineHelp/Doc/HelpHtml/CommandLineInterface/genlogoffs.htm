genlogoffs

      -genlogoffs:<OutputLogoFfsPath> -logopath:<LogoImagePathName> -guid:<GUID> -type:<EDK| EDK2>

 

To generate a logo module, the following commands are required:

      -genlogoffs : To specify the logo module's output file path.

      -logopath : To specify the logo file path.

      -guid : To specify the logo module's GUID.

      -type : To specify the EDK version.
