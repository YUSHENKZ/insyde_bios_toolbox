genbinffs

      -genbinffs:<OutputBinFfsPath> -binpath:<SrcBinFilePathName> -guid:<GUID> -type:<EDK| EDK2>

 

To generate a bin module, the following commands are required:

      -genbinffs : To specify the bin module's output file path.

      -binpath : To specify the source bin file path.

      -guid : To specify the bin module's GUID.

      -type : To specify the EDK version.
