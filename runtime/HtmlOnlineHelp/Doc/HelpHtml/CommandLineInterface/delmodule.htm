delmodule

      -delmodule:<SrcBIOSImageName> -guid:<GUID> -fvnum:<FVNum> [-out:<OutputPath>]

 

To delete a module, the following commands are required:

      -delmodule : To specify the source BIOS image to delete a module.

      -guid : To specify the module's GUID.

      -fvnum : To specify the number of the target FV (zero-based).

 

Use the following command to dispaly available modules(To get the GUID & FV number):

      -delmodule:<SrcBIOSImageName>

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
