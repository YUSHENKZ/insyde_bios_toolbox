repoprom

      -repoprom:<SrcBIOSImageName> -oprompath:<OpRomPathName> -opromindex:<Index id> -out:[<OutputPath>]

 

To replace the option rom, the following commands are required:

      -oprompath : To specify the option rom file path.

      -opromindex : To specify the module's index id.

 

Use the following command to display the available option roms:

      -repoprom:<SrcBIOSImageName>

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
