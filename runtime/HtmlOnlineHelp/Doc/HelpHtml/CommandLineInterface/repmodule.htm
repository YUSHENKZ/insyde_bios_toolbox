repmodule

      -repmodule:<SrcBIOSImageName> -modulepath:<ModulePathName> -fvnum:<FvNum> [-out:<OutputPath>]

 

To replace a module, the following commands are required:

      -modulepath : To specify the module file path.

      -fvnum : To specify the number of the target Fv (zero-based).

 

Use the following command to display available modules:

      -repmodule:<SrcBIOSImageName>

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
