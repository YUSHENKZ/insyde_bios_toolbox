repesrt

      -repesrt:<SrcBIOSImageName> [-esrtver:<ESRTVersion>] [-newguid:<GUID>] [-out:<OutputPath>]

 

To replace the version or GUID of ESRT, at least, one of the following commands is required:

      -esrtver : To specify the version of ESRT.

      -newguid : To specify the GUID of ESRT.

 

Use the following command to display the current version and GUID of ESRT:

      -repesrt:<SrcBIOSImageName>

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
