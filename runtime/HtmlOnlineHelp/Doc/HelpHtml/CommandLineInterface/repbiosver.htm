repbiosver

      -repbiosver:<SrcBIOSImageName> -biosver:<BIOSVersion> -pname:<ProductName> [-out:<OutputPath>]

 

To replace the BIOS version, the following commands are required:

      -biosver : To specify the new BIOS version.

      -pname : To specify the new product name.

 

Use the following command to display the current BIOS version and product name:

      -repbiosver:<SrcBIOSImageName>

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
