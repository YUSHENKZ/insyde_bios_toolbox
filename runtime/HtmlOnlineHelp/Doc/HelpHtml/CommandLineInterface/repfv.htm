repfv

      -repfv:<SrcBIOSImageName> -fvpath:<FvPathName> -fvnum:<FvNum> [-out:<OutputPath>]

 

To replace a whole firmware volume, the following commands are required:

      -fvpath : To specify the firmware volume file path.

      -fvnum : To specify the number of the target Fv (zero-based) to insert.

 

The following command is optional:

      -out : To specify the BIOS image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., <SrcBIOSImageName>).
